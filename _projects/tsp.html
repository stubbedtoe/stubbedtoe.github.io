--- id: tsp title: TSP Route Verifier categories: - personal - web - applications thumb: thumb.png images: - initial.png - good.png - bad.png medium: Google Maps -based web application --- I created a route-verifying web application for the Maynooth University Algorithms and Data Structures Travelling Salesman optimisation problem which is run every year. The class previously used a Java web applet for this purpose. It uses the Google Maps API to give visual feedback such as drawing the route on a map of Ireland and colouring the background to match valid/invalid routes. It also lists the problems with invalid routes and calculates the distance of any route entered. It has been approved for use by the class and can be visited here.
